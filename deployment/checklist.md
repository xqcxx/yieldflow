# Deployment Checklist

## Pre-Deployment
- [ ] Review all contract changes
- [ ] Run full test suite
- [ ] Verify code audit completion
- [ ] Check gas/fees estimation
- [ ] Notify stakeholders

## Deployment Steps
1. **Configuration**
   - [ ] Update Mainnet.toml with deployer mnemonic
   - [ ] Verify network settings
   - [ ] Test connection to Stacks API

2. **Dry Run**
   - [ ] Generate deployment plan
   - [ ] Review contract addresses
   - [ ] Verify deployment costs

3. **Execution**
   - [ ] Execute deployment command
   - [ ] Confirm transactions
   - [ ] Wait for confirmation

4. **Verification**
   - [ ] Check contract on explorer
   - [ ] Run post-deployment tests
   - [ ] Update frontend addresses
   - [ ] Monitor for 24 hours

## Post-Deployment
- [ ] Document deployed addresses
- [ ] Update README with mainnet addresses
- [ ] Announce deployment to users
- [ ] Set up monitoring
