# Deployment Checklist

## Pre-Deployment
- [ ] Code freeze
- [ ] Final tests passing
- [ ] Security audit complete
- [ ] Testnet verified
- [ ] Wallet funded

## Configuration
- [ ] Mainnet.toml configured
- [ ] Mnemonic secured
- [ ] RPC endpoints verified

## Execution
- [ ] Dry run on testnet
- [ ] Generate deployment plan
- [ ] Review transaction costs
- [ ] Execute deployment
- [ ] Confirm transactions

## Post-Deployment
- [ ] Verify on explorer
- [ ] Run integration tests
- [ ] Update frontend config
- [ ] Announce to users
- [ ] Monitor for 24 hours
